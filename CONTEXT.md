# Context

## Glossary

### Hint Bank
ชุดข้อมูลโปเกมอนที่มีคำใบ้ ชื่อไทยหลัก และชื่อไทยทางเลือก ใช้เป็นฐานข้อมูลคำถามของเกม

### Playable Pokemon
โปเกมอนที่ผู้เล่นสามารถเจอในคำถามได้ ต้องเป็นโปเกมอนที่มีอยู่ใน Hint Bank เท่านั้น

### Primary Answer
ชื่อ canonical ของโปเกมอนในระบบ ใช้เป็น key ภาษาอังกฤษใน Hint Bank เช่น `pikachu`, `bulbasaur`, `charmander`

### Accepted Answer
คำตอบที่ระบบยอมรับว่าถูก ต้องตรงกับชื่ออังกฤษ ชื่อไทยหลัก หรือ Thai Alias ของ Playable Pokemon ตัวนั้น

### Thai Alias
ชื่อไทยทางเลือกหรือชื่อเรียกที่ผู้เล่นอาจใช้ตอบ เช่น `ปิกะจู`, `พิคาชู`

### Answer Display
รูปแบบการแสดงคำตอบหลังตอบถูกหรือกดเฉลย ใช้เฉพาะชื่ออังกฤษและชื่อไทยหลัก เช่น `Pikachu / ปิกาจู` โดยไม่แสดง Thai Alias ทั้งหมด

### Round
ชุดคำถามหนึ่งรอบที่สุ่มจาก Playable Pokemon

### Revealed Hint
คำใบ้ที่ถูกเปิดแล้วในคำถามปัจจุบัน
